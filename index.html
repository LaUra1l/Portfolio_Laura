MORSLANDER

Platforma wspierająca naukę kodu Morse'a. Program zawiera tłumacza kodu, opis historii powstania tej formy komunikacji, quizy, system tworzenia własnego słownika wspierającego naukę. Kod napisany w językach JavaScript, CSS, HTML z uzyciem baz danych Firebase
HAZLO

Kompleksowa aplikacja internetowa do zlecania i realizacji zadań. Umożliwia dodawanie, usuwanie oraz zatwierdzanie lub odrzucanie wykonania przez zleceniodawcę, wraz z informacją zwrotną dla wykonawcy. Walidacja odbywa się m.in. przez dodanie zdjęcia na serwer. System obsługuje wiele powiązanych kont — jednego zleceniodawcę i wielu wykonawców. Wbudowana integracja z Gmailem zapewnia powiadomienia o aktualnościach. Aplikacja działa wyłącznie lokalnie na serwerze. (okrojona wersja kodu na profilu github po wejściu w link)
